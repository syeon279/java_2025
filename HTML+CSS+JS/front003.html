1. font 속성

      1. 글자색 : color
      2. 글자사이즈 : font-size
      3. 기울이기 : font-style:itatlic / nomal
      4. 굵게 : font-weight:bold / nomal
      5. 글꼴 : font-family

2. 문단 속성

      1. 행간 : line-height
      2. 정렬 : text-align 
      3. 들여쓰기 : text-indent 
      4. 밑줄 : text-decoration : underline(밑줄) / none / line-through 
      5. 자간(글자간격) : letter-spaceing
